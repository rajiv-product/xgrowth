üì™ Interstitial Ad

This guide provides instructions for loading and displaying interstitial ads in your application. Interstitial ads are full-screen ads that cover the interface of their host app.

Step 1: Create AdRequestConfiguration

Initialize the ad request with your placement name:

Java
Kotlin
Java 
AdRequestConfiguration.Builder configuration =
    AdRequestConfiguration.Companion.builder(context, "Your_placement_name");
Kotlin 
val configuration = AdRequestConfiguration.builder(context, "Your_placement_name")

Step 2: Load the Ad

Use the AdSterAdLoader to load the interstitial ad with appropriate listeners:

Java 
AdSterAdLoader.Companion.builder()
    .withAdsListener(new MediationAdListener() {
        @Override
        public void onInterstitialAdLoaded(@NonNull MediationInterstitialAd ad) {
            // Show interstitial ad here
        }

        @Override
        public void onFailure(@NonNull AdError adError) {
            // Handle failure
        }
    })
    .withInterstitialAdEventsListener(new InterstitialAdEventsListener() {
        @Override
        public void onAdClicked() { }

        @Override
        public void onAdImpression() { }

        @Override
        public void onAdOpened() { }

        @Override
        public void onAdClosed() { }

        @Override
        public void onAdRevenuePaid(double revenue,
            String adUnitId, String network) { }
    })
    .build()
    .loadAd(configuration.build());

Step 3: Display the Ad

Call showAd(activity) within the success callback:

Java 
@Override
public void onInterstitialAdLoaded(@NonNull MediationInterstitialAd ad) {
    ad.showAd(activity);
}

Important

Event Listeners

The InterstitialAdEventsListener supports the following callbacks:

Callback Description
onAdClicked() User clicked on the ad
onAdImpression() Ad impression recorded
onAdOpened() Ad was opened
onAdClosed() Ad was closed
onAdRevenuePaid() Provides revenue and network information
