üì™ Banner Ad

This guide provides instructions for loading and rendering banner advertisements in your application using the AdSter SDK.

Step 1: Create AdRequestConfiguration

Initialize the configuration with your placement name:

Java
Kotlin
Java 
AdRequestConfiguration.Builder configuration =
    AdRequestConfiguration.Companion.builder(context, "Your_placement_name");
Kotlin 
val configuration = AdRequestConfiguration.builder(context, "Your_placement_name")

Step 2: Load the Ad

Call loadAd() with listeners for handling callbacks:

Java
Kotlin
Java 
AdSterAdLoader.Companion.builder()
    .withAdsListener(new MediationAdListener() {
        @Override
        public void onBannerAdLoaded(@NonNull MediationBannerAd ad) {
            // Show banner ad here
        }

        @Override
        public void onFailure(@NonNull AdError adError) {
            // Handle failure here
        }
    })
    .withAdsEventsListener(new AdEventsListener() {
        @Override
        public void onAdClicked() { }

        @Override
        public void onAdImpression() { }

        @Override
        public void onAdRevenuePaid(double revenue,
            @NotNull String adUnitId, @NotNull String network) { }
    })
    .build()
    .loadAd(configuration.build());
Kotlin 
AdSterAdLoader.builder()
    .withAdsListener(object : MediationAdListener() {
        override fun onBannerAdLoaded(ad: MediationBannerAd) {
            // Show banner ad here
        }

        override fun onFailure(adError: AdError) {
            // Handle failure here
        }
    })
    .withAdsEventsListener(object : AdEventsListener() {
        override fun onAdClicked() { }
        override fun onAdImpression() { }
        override fun onAdRevenuePaid(revenue: Double, adUnitId: String, network: String) { }
    })
    .build()
    .loadAd(configuration.build())

Step 3: Display the Ad

Retrieve the view from the loaded ad and add it to your layout:

Java 
container.removeAllViews();
container.addView(ad.getView());

Step 4: Cleanup

Invoke destroy() when the activity or fragment is detached:

Java 
ad.destroy();

Key Callbacks

Callback Description
onBannerAdLoaded Ad successfully loaded
onFailure Ad loading failed
onAdClicked User clicked advertisement
onAdImpression Ad impression tracked
onAdRevenuePaid Revenue data provided with network information
