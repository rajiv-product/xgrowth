üì™ App Open Ad

The AdSter SDK provides methods to load and display App Open ads, which are a proprietary Google Ad Manager and AdMob ad format. App Open ads appear when users open or return to your app.

Step 1: Create Ad Request Configuration

Kotlin 
val configuration = AdRequestConfiguration.builder(context, "Your_placement_name")

Step 2: Load App Open Ad

Kotlin 
AdSterAdLoader.builder()
    .withAdsListener(object : MediationAdListener() {
        override fun onAppOpenAdLoaded(ad: MediationAppOpenAd) {
            super.onAppOpenAdLoaded(ad)
            // Show App Open Ad here
        }

        override fun onFailure(adError: AdError) {
            // Handle failure callback here
        }
    })
    .withAppOpenAdEventsListener(object : AppOpenAdEventsListener() {
        override fun onAdClicked() { }
        override fun onAdClosed() { }
        override fun onAdImpression() { }
        override fun onAdOpened() { }
        override fun onFailure(adError: AdError?) { }
        override fun onAdRevenuePaid(revenue: Double, adUnitId: String, network: String) { }
    })
    .build()
    .loadAd(configuration.build())

Step 3: Display App Open Ad

Within the onAppOpenAdLoaded callback, invoke the display method:

Kotlin 
override fun onAppOpenAdLoaded(ad: MediationAppOpenAd) {
    super.onAppOpenAdLoaded(ad)
    ad.showAd(activity)
}

Callback Events

The SDK provides event listeners for tracking ad interactions:

Callback Description
onAdClicked When users interact with the ad
onAdImpression When ad displays
onAdOpened When ad opens
onAdClosed When ad closes
onFailure Error handling
onAdRevenuePaid Revenue tracking with network source
